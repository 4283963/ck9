## 1. 产品概述

工业晶圆多层微观结构 3D 缺陷可视化系统，用于在网页端直观展示晶圆各层级的三维结构与缺陷分布，帮助工程师快速定位缺陷所在层级和空间位置。
- 目标用户：半导体制造工程师、良率分析人员、质量控制团队
- 核心价值：将平面化的晶圆层级缺陷数据转化为可交互的 3D 可视化，缩短缺陷定位时间，提升良率分析效率

## 2. 核心功能

### 2.1 用户角色
| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 工程师 | 直接访问 | 浏览 3D 可视化、交互操作、查看缺陷详情 |

### 2.2 功能模块
1. **3D 晶圆可视化页**：多层圆盘模型、缺陷红球标注、旋转缩放交互、层级信息面板
2. **缺陷详情面板**：缺陷类型、坐标信息、所在层级

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 3D 晶圆可视化页 | 多层圆盘渲染 | 将晶圆各层渲染为半透明圆盘，层间有间距，支持 OrbitControls 旋转缩放 |
| 3D 晶圆可视化页 | 缺陷标注 | 在缺陷三维坐标处渲染红色球体，球体大小可区分缺陷类型 |
| 3D 晶圆可视化页 | 层级控制面板 | 左侧面板列出所有层级，可点击高亮某层、隐藏/显示某层 |
| 3D 晶圆可视化页 | 缺陷信息面板 | 点击红球后显示缺陷详情（类型、坐标、层级） |
| 3D 晶圆可视化页 | 统计概览 | 顶部展示总缺陷数、各类型缺陷数、各层缺陷数 |

## 3. 核心流程

用户打开页面 → 前端请求后端 API 获取晶圆数据 → 渲染多层圆盘 3D 模型 → 在缺陷坐标绘制红球 → 用户旋转/缩放查看 → 点击红球查看缺陷详情 → 通过层级面板筛选关注层

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深色科技蓝（#0a1628）背景，搭配亮青色（#00e5ff）和警示红（#ff3d3d）强调色
- 按钮风格：圆角微透明按钮，带发光边框
- 字体：Rajdhani（数据展示）+ Noto Sans SC（中文内容）
- 布局风格：全屏 3D 画布 + 左侧浮动面板 + 右下角信息浮层
- 图标风格：线性科技感图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 3D 晶圆可视化页 | 3D 画布 | 深色背景、半透明蓝色圆盘层、红色发光球体、OrbitControls 交互 |
| 3D 晶圆可视化页 | 左侧层级面板 | 半透明深色面板、层级列表项、可见性切换开关、选中高亮态 |
| 3D 晶圆可视化页 | 顶部统计栏 | 缺陷总数、类型分布、层级分布的数字卡片 |
| 3D 晶圆可视化页 | 缺陷详情弹窗 | 坐标信息、缺陷类型标签、层级归属 |

### 4.3 响应式设计
- 桌面端优先设计，3D 画布全屏
- 面板在窄屏下可折叠

### 4.4 3D 场景引导
- 环境：深色太空感背景，带微弱网格地面
- 灯光：环境光 + 方向光，营造冷调工业氛围
- 相机：透视相机，初始俯视 30° 角度，OrbitControls 自由旋转
- 构图：晶圆模型居中，各层等距排列
- 交互：鼠标拖拽旋转、滚轮缩放、点击红球弹出信息
- 后处理：红球带辉光（UnrealBloomPass）效果，增强缺陷视觉冲击
